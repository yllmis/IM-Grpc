# IM 前端练习工程（Vue3）

## 你会先学到什么
- 前端的本质是状态映射：把后端数据状态映射到可交互 UI。
- IM 最小闭环是：登录 -> 拉会话 -> 建立 WS -> 收发消息。
- 先保证数据链路正确，再做复杂样式和动画。

## 当前工程说明
- 本工程只提供可运行骨架。
- 关键业务逻辑留有 TODO，由你逐步完成。
- 每次补完一块后，建议手动验证再继续下一块。

## 快速启动
1. 安装依赖

```bash
npm install
```

2. 启动开发环境

```bash
npm run dev
```

## 第一阶段练习任务（建议按顺序）
1. 在 src/api/http.js 中补齐 token 注入与统一错误处理。
2. 在 src/views/LoginView.vue 中接入真实登录接口并存储会话。
3. 在 src/views/ChatView.vue 中拉取会话列表并渲染。
4. 在 src/ws/imSocket.js 基础上补齐断线重连与心跳。
5. 在聊天页接入发送消息与接收消息渲染。

## 第一性原理提示
- 任何前端功能都可以拆成三件事：输入、状态、输出。
- 例如登录：输入是账号密码，状态是 token，输出是页面跳转和登录态。
- 例如消息收发：输入是文本和发送动作，状态是消息列表，输出是消息区渲染。
